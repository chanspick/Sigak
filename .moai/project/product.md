# SIGAK — 제품 개요

## 프로젝트 정의

**SIGAK**은 AI 기반 퍼스널 이미지(PI) 진단 서비스입니다. 얼굴 구조 분석, 미감 좌표 매핑, AI 리포트 생성을 결합하여 사용자가 자신의 미적 포지션을 객관적으로 이해하고 원하는 방향으로 최적화할 수 있도록 돕습니다.

**한 줄 요약:** "당신을 아는 사람들. 당신이 온전히 당신일 수 있게."

## 핵심 기능

### 1. 랜딩 페이지 (/)
- 예약 캘린더 (21일 x 10슬롯 x 3티어 = 630석)
- 3개 서비스 티어: 시선(₩5K), Creator(₩200K), Wedding(₩200K)
- 예약 오버레이 (티어 -> 날짜 -> 시간 -> 폼 -> 결제)

### 2. 인터뷰 대시보드 (/dashboard)
- 대기열 관리 (예약일시/상태별)
- 인터뷰 데이터 입력 (코어 6개 + 티어별 추가 질문)
- 가설 검증 지표 (H1 시장, H2 제품, H4 성장)
- 결제 확인 뷰 (pending 목록 + 확인/미확인)

### 3. 리포트 뷰어 (/report/[id])
- 8개 섹션 렌더링 (cover ~ trend_context)
- 블러 페이월 시스템 ("답이 있다는 것은 보여주고, 답 자체는 가린다")
- 5단계 access_level (free/standard_pending/standard/full_pending/full)
- 결제 -> pending -> 알바확인 -> 잠금해제 플로우

### 4. CV 분석 파이프라인
- MediaPipe 468포인트 얼굴 랜드마크
- 11개 구조적 메트릭 + 피부톤 분석
- 4축 미감 좌표계 (구조/인상/성숙도/강도)

### 5. AI 리포트 생성
- Claude API 기반 인터뷰 해석 -> 추구미 좌표
- 6섹션 구조화 리포트 + 실행 가이드
- WeasyPrint PDF 렌더링

## 디자인 철학

### 브랜드 톤
- **미니멀 에디토리얼**: 콘텐츠 자체로 말하는 디자인
- **따뜻한 전문성**: "판단이 아닌 분석"
- **프리미엄 절제**: 과시보다 신뢰

### 비주얼 시스템
- **컬러:** #F3F0EB(웜 아이보리) + #000000(블랙) 듀오톤
- **타이포:** Noto Serif KR(헤드라인/수치) + Pretendard Variable(본문/UI)
- **레이아웃:** 3컬럼 그리드(1:1:2), 1px 디바이더, 60px 패딩
- **인터랙션:** opacity 호버, 스크롤 reveal, 슬라이드 오버레이
- **블러:** backdrop-filter: blur(12px) + rgba(243,240,235,0.7)
