문제가 바로 보인다.
광고 2번 ("셀카 올리고 캐스팅")에서 클릭한 유저가 sigak.asia에 도착하면 첫 화면이 "미감을 좌표로 읽다"야. "좌표? 나 캐스팅 기회 보러 왔는데?" — 이 순간 이탈이야. 광고의 약속과 랜딩의 첫인상이 완전히 다른 거지.
현재 페이지 구조가:
1. Hero: "미감을 좌표로 읽다" ← 분석 앵글
2. About: "라벨이 아닌 좌표를 드립니다" ← 분석 앵글
3. Report: 리포트 미리보기 ← 분석 앵글
4. Method: 3축 설명 ← 분석 앵글
5. Process: 셀카→리포트 ← 분석 앵글
6. Casting: "분석을 넘어, 기회로 연결" ← 여기서야 캐스팅
7. Team
   캐스팅이 6번째 섹션이야. 2번 광고 타고 온 유저는 거기까지 스크롤 안 해.
   근데 여기서 중요한 판단이 하나 있어. 홈페이지를 통째로 캐스팅 앵글로 바꿀 거냐, 아니면 광고별 랜딩을 분리할 거냐.
   통째로 바꾸면 — 캐스팅 기대감으로 들어온 유저한테는 좋은데, 실제 상품은 ₩49,000 리포트잖아. "캐스팅 받을 수 있어요!"로 끌어놓고 "리포트 사세요"가 나오면 기대 갭이 생겨. 이건 전환은 올릴 수 있지만 환불/불만이 올라갈 리스크가 있어.
   내 제안은 이래:
   광고 2번 전용 랜딩을 /casting으로 따로 만들어. 기존 홈은 유지하되, 2번 광고의 링크를 sigak.asia/casting으로 건다. 이 페이지의 스토리 흐름이:
   Hero: "셀카 한 장으로 캐스팅 제안을 받아보세요"
   → 도파민 훅. 핵심 약속이 첫 화면.
   How it works:
    1. 셀카 업로드 + 5분 설문
    2. AI가 당신의 매력 포인트를 분석
    3. 에이전시가 좌표 기반으로 탐색
    4. 캐스팅 제안이 도착
       Social proof:
       → 캐스팅 제안 목업 카드 (이미 있는 UI 재활용)
       → "Alpha Agency에서 화보 촬영 제안 ₩500,000"
       But first:
       "캐스팅 풀에 등록하려면 먼저 AI 진단을 받아야 해요"
       → 여기서 리포트 구매로 자연스럽게 연결
       → ₩5,000 오버뷰 or ₩49,000 풀 리포트(현재 할인중 가격 2,900, 29,000)
       → ₩100,000 셀럽 풀 (우선 매칭)
       CTA: "내 매력 분석 시작하기"

이러면 유저 심리 흐름이: 캐스팅 기회 보러 왔어 → 어떻게 하는 건데? → AI가 내 매력을 분석해주고 → 에이전시가 찾아준대 → 일단 진단받아볼까 → ₩2,900이면 해볼 만하지
핵심은 리포트 구매가 캐스팅의 전제 조건으로 프레이밍되는 거야. "리포트 사세요"가 아니라 "캐스팅 풀 등록을 위해 먼저 AI 분석이 필요해요"로. 실제로도 opt-in이 풀 리포트 이후에 나오니까 거짓말도 아니고.
기존 홈(/)은 1번 광고용 + 오가닉 유입용으로 그대로 두고, /casting 랜딩만 새로 만들면 작업량도 크지 않아. 기존 컴포넌트(캐스팅 제안 카드, 프로세스 스텝) 재활용하면 되니까.
