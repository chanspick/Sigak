## 공통 프롬프트 규칙

```
기본 스타일:
- 한국 여성, 20대, 스튜디오 조명
- 다크 배경 (#1A1A1E) — SIGAK 브랜드 컬러와 통일
- 피부톤: 자연스러운 한국인
- 표정: 무표정~미소 (뷰티 촬영 톤)
- 저작권 free AI 생성

헤어 레퍼런스 (H-시리즈):
- 정면 또는 3/4 각도
- 상반신 크롭 (어깨선까지)
- 동일 배경, 동일 조명
- 각 스타일의 핵심 형태감이 명확히 드러나도록

메이크업 레퍼런스 (M-시리즈):
- 눈썹: 반얼굴 3/4 클로즈업
- 아이: 눈 영역 클로즈업
- 블러셔: 정면 풀페이스
- 쉐딩: 정면 + 반측면
- 립: 입술 영역 클로즈업

블러셔/쉐딩 영역 오버레이:
- AI 베이스 이미지 위에 반투명 컬러 영역 합성
- 블러셔: 핑크/코랄 (#FF6B8A, opacity 40%)
- 쉐딩: 브라운 (#8B6914, opacity 30%)
- 하이라이터: 화이트 (#FFFFFF, opacity 25%)
- 영역 경계: 점선 (#FFFFFF, opacity 60%)
- 영역 라벨: a, b, c (흰색, 12pt)
```
