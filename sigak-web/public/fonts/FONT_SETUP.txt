Pretendard Variable 폰트 설정 안내
===================================

이 디렉터리에 다음 파일을 추가해야 합니다:

  PretendardVariable.woff2

다운로드 경로:
  https://github.com/orioncactus/pretendard/releases

설치 방법:
  1. 위 GitHub 릴리스 페이지에서 PretendardVariable.woff2 파일을 다운로드합니다.
  2. 다운로드한 파일을 이 디렉터리 (public/fonts/)에 배치합니다.
  3. app/layout.tsx에서 localFont로 참조되므로 파일명이 정확해야 합니다.

참고:
  - 폰트 파일이 없을 경우 시스템 폰트 폴백이 적용됩니다.
  - CDN 폰트 사용은 금지되어 있으므로 반드시 로컬 파일을 사용합니다.
