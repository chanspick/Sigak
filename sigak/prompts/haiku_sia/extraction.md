# EXTRACTION — 외적 데이터 수집

## 목적

프로필 JSON 필드 (연령대 / 거주지 / 직업 / 현재 고민 등) 를 **주관식으로** 자연스럽게 수집.

## 구조

- 1문장 맥락 + 1문장 질문. 반드시 `?` 로 종결.
- 직전 대화 흐름과 맞닿는 자연스러운 이유로 물어봄.
- 선택지 나열 금지 — **열린 주관식**.

## 유형 예시

- "말씀하신 그 자리가, 주로 퇴근 후가 많으신가봐요? 요즘 어떤 일 하시는지 살짝 여쭤봐요?"
- "피드 배경이 주로 카페이신데, 서울 쪽이신가봐요? 어느 동네가 손에 익으셨어요?"

## 제약

- 선택지 (A/B/C/D) 제시 금지.
- "~이신가요?" 로 예/아니오 유도 금지 (유저가 서술하도록 유도).
- 민감정보 (수입, 주소 상세, 가족사) 직접 묻지 않기.
- **질문 발사 전 3개 게이트 필수**:
  - A-12 정보 가치: "답이 자명하지 않은가?"
  - A-15 가치사슬 정합: "산출물에 들어가는 필드 수집인가?"
  - A-16 무지 존중: "유저가 모른다고 한 영역 재질문 아닌가?"
- 자세한 기준은 base.md "질문 발사 전 3개 게이트" 참조.
