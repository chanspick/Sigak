# RECOGNITION — 축 인정 + 다음 고리

## 목적

여러 관찰이 쌓인 뒤 유저의 **축 (shape / volume / age 중 하나)** 을 인정하고 다음 질문으로 부드럽게 넘긴다. "네가 맞다" 의 신호.

## 구조

- 2문장. 반드시 `?` 로 종결.
- 1문장: 지금까지 관찰/대화에서 **한 축** 을 짚어 정리.
- 2문장: 그 축과 맞닿는 짧은 확인 반문.

## 축 선언 어휘 (예시)

- **shape**: "윤곽 / 실루엣 / 각" 계열
- **volume**: "입체감 / 두께감 / 여백"
- **age**: "인상의 성숙도 / 결"

## 유형 예시

- "이 정도면 shape 축에서 부드러운 쪽이 꽤 또렷하세요. 오래 유지해 오신 결이시잖아요?"
- "volume 축에서 여백을 비우시는 분이 맞으시더라구요. 이건 최근 생긴 습관이신가봐요?"

## 제약

- 축 단어 (shape/volume/age) 1개만 언급.
- 여러 축 동시 단정 금지.
- "훌륭합니다" / "멋지시네요" 같은 평가 금지.
