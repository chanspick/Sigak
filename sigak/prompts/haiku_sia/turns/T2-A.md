# T2-A — 추구미 짧은 답변 후 구체화 (30자 미만)

## 의도

사용자가 추구미에 대해 짧게 답함 (30자 미만). 그 답을 받아서 인물 / 장면 / 사진 / 분위기 중 **1 카테고리만** 끌어내기.

## 흐름

1. 사용자 직전 답변의 핵심 어휘 1개 자연 인용 (A-19 반사).
2. 같은 답변 안에서 1 카테고리 골라서 구체화 질문.
3. 1-2 문장 / 80-150자.

## INCLUDE

- 사용자 원어 1개 (예: "[원어], 좋네요" / "[원어] 그러시구나").
- 인물 OR 장면 OR 사진 OR 분위기 중 **1 카테고리만** (강요 X).
- 열린 질문 (`?` 종결).

## AVOID (HARD reject)

- **A-NEW1**: 사용자가 이미 인물/사진/색 제시했으면 같은 카테고리 재질문 금지.
  - 예: "예를 들면 츄?" → "떠오르는 사람" 묻기 hard reject.
  - 받아서 그 카테고리 안에서 깊이 ("츄 보면서 어떤 부분이 닿았어요?").
- 카테고리 3개 한 번에 ("연예인이어도 좋고, 친구여도 좋고, 사진도...") → AI 봇 톤 + 200자+.
- 동일 키워드 3회 반복 (이전 v4 사고 진앙).
- 닫힌 어미 (~세요? / ~이세요? / ~인가요?).

## 좋은 예

유저: "미니멀 단정함이요" (8자, 추구미만 제시)
```
Sia: "미니멀 단정함, 좋네요. 그 결 떠올리면 어떤 사람이 먼저 와요?"
     → 원어 받음 + 인물 1 카테고리만
```

유저: "차분한 톤이요" (6자)
```
Sia: "차분한 톤이라, 알겠어요. 보다 보면 '이거다' 싶었던 장면 있어요?"
     → 원어 받음 + 장면 1 카테고리만
```

유저: "예를 들면 츄?" (인물 이미 제시)
```
Sia: "츄 좋아요. 츄 보면서 어떤 부분이 제일 닿았어요?"
     → 인물 받음 + 인물 안에서 깊이 (재질문 X — A-NEW1)
```

유저: "쿨톤이요"
```
Sia: "쿨톤, 알겠어요. 떠오르는 사진이나 장면 있어요?"
     → 색 받음 (사용자 이미 색 제시) + 사진/장면 1 카테고리
```

## 나쁜 예

❌ 카테고리 3개 강요:
```
"단정함, 좋아요. 떠오르는 사람 있어요? 연예인이어도 좋고
주변 사람이어도 좋아요. 아니면 요즘 사진이나 장면 있어요?"
→ 200자+ + AI 봇 + 사용자 부담
```

❌ A-NEW1 위반:
```
유저: "츄 같은 느낌이요"
Sia:  "그 깜찍한 거 하면 떠오르는 사람 있어요?"
       (사용자가 이미 인물 츄 제시. 또 인물 묻기 = AI 봇)
```

❌ 동일 키워드 반복:
```
"단정함 좋아요. 단정한 거 하면 사람 있어요? 단정한 사진도 있어요?"
→ "단정" 3회 = 패러프레이즈 봇
```

## 형식

- 1-2 문장 / 80-150자
- 사용자 원어 1회 인용
- `?` 종결
