# T4 — 추구미 → 피드 전환

## 의도

T1-T3 에서 추구미 영역 충분히 봤음. 이제 **피드 (현재)** 로 시선 전환. "잘 따라가고 있는지" 자기 평가 끌어내기 (T5 분기 트리거).

## 흐름

1. 전환 멘트 1 문장 ("이제 본인 피드 떠올려보세요" / "본인 피드 그려보세요").
2. 자기 평가 질문 ("잘 따라가고 있어요?" / "추구미랑 가까워요?").
3. 2 문장 / 60-90자.

## INCLUDE

- "이제" / "지금" / "본인 피드" — 전환 신호.
- "추구미를 잘 따라가" / "추구미랑 가까워" 류 자기 평가 질문.
- 단순 yes/no 답변 가능 OK (T5 에서 분기 처리).

## AVOID (HARD reject)

- **Sia 가 먼저 평가** ("도윤님 피드는 추구미와 차이 있어요" — Vision 부분 공개는 T6 예약).
- **비교 강요** ("어떤 부분이 다른가요?" — 너무 빠름, T6 격차 예약).
- **추구미 영역 재진입** (이미 끝).
- "피드 봤는데" / "피드 좀 돌아봤어요" 류 — Sia 가 본 것 공개 X (T6 예약).

## 좋은 예

```
"이제 본인 피드 한 번 떠올려보시고요.
지금 추구미를 잘 따라가고 있어요?"
```

```
"본인 피드 한 번 그려보세요.
추구미랑 지금 사이가 어때요? 가깝다, 멀다, 그 사이 어디쯤이에요?"
```

```
"이제 피드 쪽으로 가볼게요.
{user}님이 보기에는 추구미 결로 잘 가고 있어요?"
```

## 나쁜 예

❌ Sia Vision 단정 (T6 예약 영역 침범):
```
"도윤님 피드 봤는데 추구미랑 좀 멀어요."
→ Vision 단정 (T6 격차 흘리기 예약). T4 단계 위반.
```

❌ 비교 강요:
```
"어떤 부분이 추구미랑 다르게 보여요?"
→ 너무 빠름. T6 격차 영역.
```

❌ 추구미 재진입:
```
"추구미 한 번 더 정리하면 뭐예요?"
→ T3 영역 재진입. T4 는 전환이지 정리 아님.
```

❌ 사용자 답변 부담:
```
"본인 피드 모든 사진 떠올려보고, 추구미랑 일치하는 부분, 다른 부분, 모호한 부분 각각 들려주세요."
→ 200자+ + 답변 부담 + Sia 가 분석 강요
```

## 형식

- 2 문장 / 60-90자
- 전환 + 자기 평가 질문
- `?` 종결
